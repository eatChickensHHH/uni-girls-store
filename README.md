
=========
+ 前台采用UniAPP框架开发，跨平台多端部署，包括：
+ + 安卓APP
+ + 苹果APP
+ + 微信小程序(含QQ小程序)
+ + 百度智能小程序
+ + 支付宝小程序（含钉钉小程序）
+ + 头条小程序(含抖音小程序)
+ + H5页面
+ 完全组件化，尽最大可能减少彼此依赖
+ JS采用ES6语法规范，简化代码的兼顾性能(部分代码因为特殊原因使用ES5规范)
+ 样式文件采用SCSS预处理，遵循BEM规范，可自定义主题
+ 包含常用工具函数，缩短二次开发时间
